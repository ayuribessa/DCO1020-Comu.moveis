1) DEPENDENCIAS

1.a) Python 3.7.4
1.b) Conda ou PIP para instalação de libs.

### Utilizando Conda ###
a)Linux

-No terminal digite:
    conda install -c anaconda pyqt


b)Windows
- No prompt de comando digite:
    conda install -c anaconda pyqt


### Utilizando PIP ###
a)Linux

-No terminal digite:
    python -m pip install PyQt5


b)Windows
- No prompt de comando digite:
    python -m pip install PyQt5
-----------------------------------------------------------
2) EXECUTANDO A CALCULADORA
- Para windows e Linux, com o prompt/terminal aberto na pasta raíz do projeto:
    python main_nr.py
